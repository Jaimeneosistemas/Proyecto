INFORME EXCEL

¿QUE ES EXCEL?

excel es una herramienta de ofimatica que sirve para el manejo de datos y darle funciones a estos( mover mouse a la derecha :)

¿COMO FUNCIONA EXCEL?

excel funciona con unas casillas a las cuales se les da un valor o una funcion, de forma que, las funciones pueden llegar a tener condicionales que varien el valor(click a la img :)

¿QUE SE PUEDE HACER CON EXCEL?

excel tiene un gran numero de herramientas las cuales cumplen la funcion de organizacion y manejo de datos, estas herramientas pueden ser muy utiles empleando los condicionales para este mismo objetivo
